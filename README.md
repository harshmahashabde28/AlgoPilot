# AlgoPilot

AlgoPilot is an AI-powered Competitive Programming Coach designed to help programmers improve through analytics, personalized practice plans, and contest insights.

## 🚀 Features (Planned)

- Rating Analytics
- Weak Topic Detection
- Personalized Practice Planner
- Contest Performance Analysis
- Mistake Journal
- AI Coach
